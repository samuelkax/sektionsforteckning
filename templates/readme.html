README

Det här skriptet konverterar en Schneider-programmering till en sektionsförteckning som man kan använda för OR/SR samt genererar en fil för import till mbiz

Skriptet exkluderar enheter som inte ska vara med i sektionsföteckning. För närvarande är det dessa enheter:

    {% for device_type in invalid_types %}
  • {{ device_type }}
    • {% endfor %}

För konverteringen till mbiz behöver scriptet veta vad enhetstyperna i schneider-programmeringen har för artikelnummer i mbiz. Den informationen ligger i en hårdkodad fil.
Kontakta Samuel Kax för att lägga till enheter i den listan.

Hur du använder skriptet

  1. Tryck på "Bläddra" och leta upp din csv-fil som är exporterad från Schneider.
  2. Tryck på "Generera filer"
  3. Sektionsförteckning presenteras i en tabell och du kan välja att ladda ned en ny csv genom att klicka på "Ladda ned sektionsförteckning"
  4. För att ladda ner fil för import till mbiz klicka på "Ladda ned fil för mbiz

CSV-Filen

  • Filen måste vara i CSV och exporterad från Schneider
  • Skriptet kollar detta genom att säkerställa att rad två innehåller "Rapport konfiguration"
  • Skriptet filtrerar bort enheter som inte ska vara med. Kontakta Samuel Kax för att lägga till enheter i den listan.
